# Whimsy Forest English V5.3
校園森林對話練習版

## 重建原則
- 以現有功能與題庫為基礎更新介面，不打掉重做
- 首頁、地圖、課程卡片、進度面板皆維持可操作的 HTML / CSS 元件
- 使用校園森林背景與三位鳥類引導角色
- 所有文字、按鈕、課程內容都是真正可操作的網頁元件

## 功能
- 基礎、進階、挑戰三個對話階段，不綁定年級，可依學生目前能力自由選擇
- 喜鵲、五色鳥、貓頭鷹三位校園鳥類引導角色
- 校園森林入口
- CSS 原生故事式學習地圖
- 30 堂由簡入深的對話課程
- Listen / Repeat / Role Play
- 語音朗讀、慢速播放、錄音辨識
- XP、星星、徽章與完成課程紀錄
- 手機、平板與桌機響應式介面

## 使用
解壓縮後，直接雙擊 `index.html`。
建議使用 Chrome 或 Edge。
