# Whimsy Forest English V4.1
原生 UI 重建版

## 重建原則
- 不直接插入任何整頁設計概念圖
- 首頁、地圖、課程卡片、進度面板皆以 HTML / CSS 建立
- 圖片只保留三位獨立吉祥物素材
- 所有文字、按鈕、課程內容都是真正可操作的網頁元件

## 功能
- 低、中、高年級三條學習路線
- CSS 原生森林村莊入口
- CSS 原生故事式學習地圖
- 30 堂分級課程
- Listen / Repeat / Role Play
- 語音朗讀、慢速播放、錄音辨識
- XP、星星、徽章與完成課程紀錄
- 手機、平板與桌機響應式介面

## 使用
解壓縮後，直接雙擊 `index.html`。
建議使用 Chrome 或 Edge。
